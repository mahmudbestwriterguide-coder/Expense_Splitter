FlatSplit CLI

Overview

FlatSplit is a Rust command-line tool designed to simplify splitting expenses among roommates. It helps users calculate shares automatically, manage payments, and avoid confusion in shared apartments.

Users can input total expenses, number of tenants, and optionally, who paid what. FlatSplit then generates a clear breakdown of each tenant’s share.

How the Code Works

  • Written in Rust for performance and reliability.
  • Input: Number of tenants, total expenses, and individual contributions.
  • Process: Calculates fair share for each tenant automatically.
  • Output: Displayed in the terminal or can be saved to a file.

Example Commands

./flatsplit

Marketing Points & Strategy

  • Perfect for students and flatmates sharing living costs.
  • Reduces disputes over shared expenses.
  • Easy to use, fast, and open-source.
  • Can be extended or modified for personal use.
